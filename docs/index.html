Learnings

[anti-pattern] Modifying `.github/workflows/` files requires `workflows` scope in the GitHub App token; the agent's token only has `contents` write permission — workflow file edits must be done via a PR from a branch or by a human with appropriate permissions. [tool-usage] `gh issue close` requires GH_TOKEN to be set; it is not available during agent execution outside of a GitHub Actions workflow context. [pattern] Existing tests sometimes already cover a behavior indirectly — the `"groups items by status"` test already asserted Done exclusion; adding a dedicated, explicitly-named test is still valuable as documentation and regression pinning. [domain] ROADMAP.md housekeeping is a high-value low-risk task that directly reduces noise in the planning context fed to the LLM each cycle.
